Bootstrap Hero

A hero pattern is an oversized image that stretches across the top of the page

Usage: An attractive, oversized image immediately catches the user's attention. The key is to have an appropriate image. The size of a hero image may vary. In this template, the image takes up approximately the upper half of the viewport. A common variation of the hero image is the cover image that takes up the entire viewport.

If you use text over your image, the text should be a CSS overlay and not an actual part of the image. This template does not utilize text over the image. The cover template provides an example of using text over an image. Whenver you use text over an image, you must be concerned with contrast so that the text is readable. This usually involves adjusting the opacity of the background image or, more commonly, using a box between the text and the image. The opacity and color of the box can be adjusted for design preference. See the cover template for an example.

Use this template to kickstart your design.
